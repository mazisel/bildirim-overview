# Bildirim Servisi

**🇹🇷 Türkçe**

Supabase veritabanındaki kritik olayları anlık Telegram bildirimine dönüştüren otomasyon servisi.

## Tech Stack
- Node.js
- Supabase Realtime
- Telegram Bot API

## Özellikler
- Gerçek zamanlı veritabanı dinleme
- Filtreli ve koşullu bildirim
- Hızlı ve güvenilir mesaj iletimi
- Docker desteği

---

**🇬🇧 English**

An automation service that converts critical events in Supabase database into instant Telegram notifications.

## Tech Stack
- Node.js
- Supabase Realtime
- Telegram Bot API

## Features
- Real-time database listening
- Filtered and conditional notifications
- Fast and reliable message delivery
- Docker support
